## Featured projects

### [GlucoDesk](https://github.com/FilippoGaravaglia/GlucoDesk)

**A local-first desktop companion for glucose awareness.**

GlucoDesk is a cross-platform desktop app built with **.NET and Avalonia** to help people keep an eye on CGM glucose data while working on a computer.

It is not just a demo repository: it includes a real desktop product flow with dashboard, local history, provider status, diary export, privacy-oriented storage and public preview releases.

Key areas:

* .NET / Avalonia desktop application
* macOS and Windows support
* macOS DMG packaging
* Windows setup wizard and portable package
* macOS menu bar companion
* Windows tray companion
* Local-first history and settings
* Secure credential storage through OS-level stores
* Excel and PDF diary export
* GitHub Actions release workflow
* Professional README, release notes, safety notices and install instructions

Current public preview:

[**GlucoDesk v0.2.2-preview**](https://github.com/FilippoGaravaglia/GlucoDesk/releases/tag/v0.2.2-preview)

### [DevMemory](https://github.com/FilippoGaravaglia/DevMemory)

**A local-first .NET CLI for capturing and reusing developer knowledge.**

DevMemory is a command-line tool designed around a simple idea:

> developers lose technical context too easily.

It helps capture technical decisions, implementation notes, refactoring context, setup steps and project knowledge as local structured memories that can be searched and reused later.

Key areas:

* .NET CLI / global tool
* Local-first developer knowledge base
* Semantic search
* RAG-oriented workflows
* Qdrant integration
* Ollama/local AI experimentation
* Git-aware memory drafting
* Timeline and graph-oriented memory exploration
* Automated tests and CI

This project explores how AI-assisted workflows can support real engineering memory, not just code generation.
